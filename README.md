## Getting Started

### installation

Татаж авсны дараа доорх командыг зөвхөн нэг удаа ажиллуулна. Дахиж ажиллуулах шаардлагагүй.

```bash
npm i
```

### Run

```bash
npm run dev
```

### Task

1. `/pages/hero/[id].js` орох үед `http://129.168.../hero/:id` API дуудаж авчирдаг болгох

Шаардагдах мэдлэг: state, fetch
